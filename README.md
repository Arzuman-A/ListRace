# Listrace Veb Layihəsi

Bu layihə "Listrace" dizayn şablonundan istifadə edilərək hazırlanmış, məlumatların kataloq şəklində təqdim edildiyi veb-saytdır.

##  Texnologiyalar
Layihə aşağıdakı texnologiyalar əsasında yığılıb:
- **HTML5** - Struktur.
- **CSS3** - Dizayn və "Listrace" üslubları.

## 📁 Layihə Strukturu
- `index.html`: Əsas səhifə.
- `listrace.css`: Saytın dizayn faylı.
- `/image`: Layihədə istifadə olunan qrafik elementlər və şəkillər.
- `/fonts`: Saytda istifadə olunan şriftlər.

## 🚀 Əsas Xüsusiyyətlər
- Responsiv dizayn (mobil cihazlara uyğun).
- Səliqəli interfeys və qrafik elementlər.
- İstifadəçi üçün rahat kataloq görünüşü.
